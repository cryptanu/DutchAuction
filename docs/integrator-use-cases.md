# Integrator Use-Cases

- One-transaction swap + auction settlement flow
- Privacy-preserving amount handling via encrypted token paths
- Preflight quote API for reliable UX

## Secondary

- Private whitelist rounds
- Batch sale windows
- Treasury disposal auctions

- Frontend launchpads: wallet-native UX, fail-fast guards, activity feed
- Backend launch infrastructure: deterministic quotes and tx orchestration
- Analytics/indexers: event-driven auction state and settlement timelines
