# Support Model (v1)

v1 is self-serve by design.

## What maintainers provide

- Stable SDK interfaces for integration
- ABI + metadata artifacts
- Reference launchpad example
- Docs and troubleshooting guides

## What integrators own

- Wallet UX and transaction policy
- `@cofhe/sdk` key/session handling
- Deployment/environment management
- Product-specific analytics and monitoring
